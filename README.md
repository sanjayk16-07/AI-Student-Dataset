# AI Student Dataset Analysis

## Overview

This project analyzes the impact of AI usage on students using Machine Learning techniques.

The project includes:

* Classification
* Regression
* Clustering
* Data Visualization

---

##Dataset

Dataset: AI Student Dataset

Contains student-related features used to analyze patterns, predict outcomes, and discover groups.

---
## Project Setup

### Requirements

```bash
pip install pandas numpy matplotlib seaborn scikit-learn tensorflow
```

### Run

```bash
Open Google Colab
Upload Dataset
Run Cells Sequentially
```
